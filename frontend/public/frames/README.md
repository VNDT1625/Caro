# Avatar Frames

Thư mục này chứa các ảnh khung avatar.

## Cấu trúc file
- `bronze.png` - Khung đồng (common)
- `silver.png` - Khung bạc (rare)
- `gold.png` - Khung vàng (epic)
- `diamond.png` - Khung kim cương (legendary)
- `fire.png` - Khung lửa (epic)
- `ice.png` - Khung băng (rare)
- `nature.png` - Khung thiên nhiên (common)
- `cyber.png` - Khung cyber (legendary)

## Yêu cầu kỹ thuật
- Kích thước: 200x200px (hoặc lớn hơn, tỷ lệ 1:1)
- Format: PNG với transparent background
- Khung phải có lỗ tròn ở giữa để hiển thị avatar
- Lỗ avatar chiếm khoảng 75% kích thước frame

## Upload qua Admin
Admin có thể upload frame mới qua:
1. Admin Panel > Shop > Thêm item mới
2. Chọn category: avatar_frame
3. Upload ảnh preview
