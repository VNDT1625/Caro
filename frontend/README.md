Frontend skeleton

Suggested stack: React + TypeScript + Vite, Three.js for 3D. Basic folders:
- `src/pages` (Home, Lobby, Room, Match, Profile, Store)
- `src/components` (Board, Tile, Timer, Chat)

To scaffold quickly:
1) Install Node.js
2) Run `npm init vite@latest frontend -- --template react-ts` from `c:\PJ\caro` (or use your preferred starter)
